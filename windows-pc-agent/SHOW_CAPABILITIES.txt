    pro──
    POST /api/v1/pc/{pc_id}/command
    {
      "command": "ldplayer_list",
      "params": {}
    }
    
    for each emulator:
      POST /api/v1/pc/{pc_id}/command
      {
        "command": "ldplayer_launch",
        "params": {"index": 0}
      }

  Сценарий 2: Установить APK на все
  ────────────────────────────────────────────────────────────────────────────
    1. Скачать APK:
       POST /api/v1/pc/{pc_id}/command
       {
         "command": "net_download",
         "params": {
           "url": "https://example.com/app.apk",
           "path": "C:\\temp\\app.apk"
         }
       }
    
    2. Установить на каждый эмулятор:
       for each emulator:
         POST /api/v1/pc/{pc_id}/command
         {
           "command": "ldplayer_install_apk",
           "params": {
             "index": 0,
             "apk_path": "C:\\temp\\app.apk"
           }
         }

  Сценарий 3: Мониторинг и автоочистка
  ────────────────────────────────────────────────────────────────────────────
    1. Проверить процессы:
       POST /api/v1/pc/{pc_id}/command
       {"command": "process_list", "params": {}}
    
    2. Если CPU > 90%, очистить:
       POST /api/v1/pc/{pc_id}/command
       {
         "command": "exec_batch",
         "params": {
           "content": "del /q C:\\temp\\*.tmp\ntaskkill /f /im chrome.exe"
         }
       }

  Сценарий 4: Автоматический деплой

    3. Обновить APK
